Tokyo Olympics 2021 - Analytics Dashboard
📘 Project Overview

This project analyzes Tokyo Olympics 2021 data using Power BI to derive insights on athletes, countries, and medal distributions.

It also forms part of a Data Engineering pipeline using Azure Data Factory, Data Lake, Synapse Analytics, and Power BI for visualization.

🧩 Tools & Technologies Used

Azure Data Factory – For data ingestion and pipeline creation

Azure Data Lake Storage – For storing raw and transformed data

Azure Synapse Analytics – For data transformation and querying

Power BI Desktop – For building the interactive dashboard

GitHub – For project version control and portfolio hosting

📊 Dashboard Insights

The Power BI Dashboard includes:

Total Entries by Gender

Total Medals Overview

Total Medals by Country

Total Players by Country

Key insights:

Top performing countries: United States, China, ROC, Great Britain

Visual comparison of Gold, Silver, Bronze medals by country

Gender distribution across Olympic participants
